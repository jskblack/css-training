CSS 訓練館

別擔心，你辦得到的！

你即將開始學習 CSS 選擇器！ 選擇器是用來選取你想要套用樣式的特定元素。

範例 1 - 一條 CSS 規則

p {
   margin-bottom: 12px;
}

在這裡，"p" 是選擇器（選取所有的 <p> 元素）並套用下外邊距樣式。

要開始遊戲，請在下方的編輯器輸入 CSS 選擇器來選取桌面上正確的項目。如果你答對了，將會進入下一關。

將滑鼠停留在桌面上的項目，可以查看它們的 HTML 標記。

在右側可以獲得選擇器的說明與幫助！ →
